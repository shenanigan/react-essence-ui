# Install the library

You can install the library using the `npm install` command. To install the library, run the following command:

terminal

    npm install @wp-essence-ui/components

# Import the component

To use the components you can import the respective components. e.g. Import a button component in your tsx file as below.

sample.tsx

    import { Button } from '@wp-essence-ui/components';

# Use the component

You can use the component as below.

sample.tsx

    <Button>Hello World</Button>

For more information visit [our website](https://essence.royalecheese.com/getting-started?utm_source=npm&utm_medium=readme&utm_campaign=essence-ui)
